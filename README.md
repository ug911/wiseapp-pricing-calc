## About This Calculator

This tool helps you compare three flexible billing models that wise.live offers, so you can pick the one that fits your business best.

### Billing Models

| Plan | How it works |
|------|-------------|
| **Per Session** | Pay based on the total number of sessions conducted each month (1:1 + group) |
| **Per Seat** | Pay based on the number of staff seats (tutoring + non-tutoring) |
| **Per Student** | Pay based on the total number of active students enrolled |

### Unit Pricing (USD)

| Type | Price |
|------|-------|
| Online session | $1.80 |
| Offline session | $0.80 |
| Online seat | $40.00 |
| Offline seat | $25.00 |
| Student (online) | $5.00 |
| Student (offline) | $2.00 |

> Prices are indicative and for estimation purposes only. [Contact wise.live](https://www.wise.live/contact/) for a tailored quote.
